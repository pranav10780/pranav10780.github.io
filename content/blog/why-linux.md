Linux is the foundation of almost everything interesting in computing —
servers, Android, embedded systems, and most of the open source world runs on it.

## Why Linux

After switching from Windows I noticed a few things immediately. The terminal
is actually useful. Software installation is one command. The system does what
you tell it to do rather than the other way around.

## Where to start

If you have never used Linux before, start with a distribution that has good
documentation and a large community. Arch Linux is not the place to start —
but the [ArchWiki](https://wiki.archlinux.org) is useful regardless of which
distribution you use.

If you don't have a pc at home i highly recommend using [Termux](https://wiki.termux.com/wiki/Main_Page) to learn linux
